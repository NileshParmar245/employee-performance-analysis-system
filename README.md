# Employee Performance Analysis System

## Project Overview

This project analyzes employee performance and productivity data to help HR departments make data-driven decisions.

## Business Problem

Organizations need to identify top performers, understand factors affecting employee productivity, and improve employee retention through data analysis.

## Objectives

* Analyze employee performance across departments
* Identify top-performing employees
* Study salary vs performance relationships
* Analyze employee satisfaction and retention
* Generate HR recommendations

## Dataset Information

* Dataset: Extended Employee Performance and Productivity Dataset
* Records: 100,000 Employees
* Features: 20 Columns
